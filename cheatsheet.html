Array 遍历

Array.prototype.map(callback(value,index,array))

callback function allows return

Array.prototype.forEach(callback(value,index,array))>

callback function don't have return

Array.prototype.some(callback(value,index,array))

至少有一个callback返回真时some()返回真,否则some()返回假

用来检测有没有符合要求的元素

Array.prototype.every(callback(value,index,array))

全部callback返回真时every()返回真,否则every()返回假

用来检查数组符不符合要求
